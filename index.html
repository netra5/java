Animal World

Objective Of This Project

The webpage titled "Animal World" features a header with the same title, followed by a paragraph describing animals. Below the paragraph, there are two square div elements representing animals. Hovering over the first div reveals an image of a cat, while moving the mouse away displays an image of a dog. Hovering over the second div reveals an image of an lion, and moving the mouse away displays an image of a Elephant. This interactive behavior allows users to explore different animal images simply by hovering over the div elements.
